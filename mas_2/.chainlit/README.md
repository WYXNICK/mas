# MAS 多智能体测试与展示

在此输入您的任务描述，系统将调用多智能体工作流执行：

- **Supervisor**：规划与调度
- **RAG Researcher**：文献检索
- **Code Dev**：代码生成与执行
- **Tool Caller**：数据分析工具（基因查询、富集、细胞类型注释等）
- **Critic**：审核与反馈

执行过程中可在界面查看各步骤的中间结果，结束后将展示最终答案（文献、代码方案、分析报告）。

示例任务：
- 写一个函数打印 hello world
- 请对单细胞数据进行标准处理、聚类与 UMAP 可视化（需在查询中指定数据路径与输出路径）
